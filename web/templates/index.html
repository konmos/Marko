Marko is a Discord bot with a plugin for everything and a focus on social features (at least that's the plan)!

Moderation

Marko comes with moderation tools such as mass message deletion, blacklisting features, and more to make an admin's life easier!

Ranking

One of Marko's core features is a rich and fully featured ranking system with xp, levels, and custom profiles...

Reddit

Thanks to Marko you can now browse reddit without even leaving the Discord client. Text and image posts are supported!

Web Search

Ever wanted to search something but didn't want to bother with opening a browser? Now you can.

Gaming Integration

Just like Discord is an app for gamers, Marko can be a bot for gamers. Steam integration is here and more is coming soon!

Powerful Configuration

Marko was built with extensibliity and configuration in mind. Per server configs allow you to enable/disable plugins, change the prefix, ignore channels, and much more.

Reminders

Never again will you forget to study for your exam catch up on your favourite show with the easy to use remindme command...

Voice Player

With hundreds of supported streaming sites, you can now listen to music, podcasts, or streams right from the Discord voice chat! Make sure to invite some friends too...
